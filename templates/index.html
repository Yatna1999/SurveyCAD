Using the SCR Script File

  1. Open AutoCAD (new or existing drawing).
  2. Go to Manage tab → Run Script, or type SCRIPT in the command bar.
  3. Browse to your Downloads folder and select survey_output.scr.
  4. AutoCAD runs all commands automatically — layers, points, labels, polylines.
  5. Type ZOOM E to see all points (the script does this automatically).
  6. Use Layer Properties Manager (LAYER) to toggle:
    • MS POINT – plus (+) markers (white)
    • POINT_NUMBER – Sr No labels (blue)
    • DESCRIPTION – description codes (blue)
    • ELEVATION – height values (yellow)
    • PLINE_* – polylines (green)
  7. Points appear as plus (+) markers (PDMODE 2). Change via PDMODE and PDSIZE.

Using the DXF File

  1. Open AutoCAD (or QCAD, LibreCAD, BricsCAD).
  2. File → Open → select survey_output.dxf.
  3. Drawing opens with all layers, points, labels, and polylines.
  4. Type ZOOM E to fit all points in view.
  5. Use Layer Properties Manager to toggle layers.
  6. Set drawing units: UNITS → match your chosen unit.

Using the DWG File

  1. Open AutoCAD.
  2. File → Open → select survey_output.dwg.
  3. DWG is the native AutoCAD format (R2018), so you don't need any imports or scripts.
  4. Drawing opens immediately with all points, labels with the proper font (SURVEY simplex.shx), and layers correctly applied.
  5. Type ZOOM E to see your entire site map.

Tips

  • Points too small? Type PDSIZE → enter a larger value → REGEN.
  • Too much text clutter? Freeze the DESCRIPTION or ELEVATION layer.
  • SCR vs DXF: SCR runs commands in your current drawing. DXF opens as a separate file.
  • Text height: For large Canadian coordinates, try 1.0 – 5.0 meters.
